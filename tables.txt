1. Жанр - таблица, описывающая жанр фильма. 
Связана с таблицей "Фильм" связью "Многие ко многим", так как у одного фильма может быть несколько жанров, а каждый жанр может быть у неограниченного количества фильмов. 
Содержит в себе поля:
	- name: поле названия жанра. Тип - строка. Имеет ограничение по длине (максимум 30 символов).

2. Фильм - таблица описывающая фильмы. Содержит в себе поля:
Связана с таблицей "Жанр" связью "Многие ко многим". 
Связана с таблицей "Сессия" связью "Один ко многим", так как в одной сессии может быть только один фильм, а фильм может быть в неограниченном количестве сессий. 
Связана с таблицей "Отзыв" связью "Один ко многим", так как в одной отзыве может быть только один фильм, а фильм может быть в неограниченном количестве отзывов. 
Содержит в себе поля:
	- name: поле названия фильма. Тип - строка. Имеет ограничение по длине (максимум 30 символов).
	- country: поле страны фильма. Тип - строка. Имеет ограничение по длине (максимум 20 символов).
	- genre: поле жанра.
	- duration: поле длительности фильма. Тип - поле времени.
	- budget: поле бюджета фильма. Тип - дробное число.
	- poster: поле изображения постера. Тип - изображение.
	- description: поле описания фильма. Тип - строка. Имеет ограничение по длине (максимум 1000 символов).
	- rating: поле рейтинга фильма. Тип - дробное число.
	- trailerUrl: поле ссылки на трейлер фильма. Тип - строка. Имеет ограничение по длине (максимум 50 символов).

Связана с таблицей "Сессия" связью "Один ко многим", так как в одной сессии может быть только один зал, а зал может быть использован в неограниченном количестве сессий. 
Содержит в себе поля:
